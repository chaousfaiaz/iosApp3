# iOSApp2 – Scavenger Hunt App

This is the MVP for Assignment 3 in my Mobile Web Development (iOS) course at triOS College.  
It demonstrates concepts from the **FruitList**, **CardFlip**, and **PhotoPicker** tutorials in SwiftUI.

Features:
- List of 10 businesses with clues  
- Card flip to photo submission  
- Progress counter + reward tiers  
- Simple Map integration  

Assignment 4 will expand this with Location API, persistence, and polish.
